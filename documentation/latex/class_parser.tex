生成 赋值表达式 对象 

赋值表达式视为 变量名 与 等于/不等表达式 进行赋值运算~\newline
 变量名 与 等于符号 可以不存在(降级为判断相等)~\newline
 若存在连等~\newline
 递归地生成多个赋值表达式~\newline
 -\/-\/-\/-\/-\/-\/-\/-\/-\/-\/-\/-\/-\/-\/---~\newline
 有一点与 逗号表达式 不同~\newline
 逗号表达式的结合性为 从左到右~\newline
 所以第二个表达式必须是比 逗号表达式 优先级更低的赋值运算~\newline
 而赋值运算结合性从右到左~\newline
 检测到等号后，右侧允许出现赋值~\newline
 所以第二个表达式依然是赋值~\newline
 -\/-\/-\/-\/-\/-\/-\/-\/-\/-\/-\/-\/-\/-\/---~\newline
